🔒 Como funciona a segurança? ▼
🔐
Encriptação Militar
Os teus dados são protegidos com AES-256-GCM — a mesma tecnologia usada por bancos, governos e serviços militares. É matematicamente impossível de quebrar sem a tua palavra-passe.
💾
Ficheiro Local
As tuas passwords ficam guardadas num ficheiro .vault no teu dispositivo — pen, computador ou telemóvel. O ficheiro é teu, está sob o teu controlo, nunca sai do teu dispositivo.
🌐
Zero Servidores
O CipherVault é apenas uma app de leitura. Não tem servidores, não tem base de dados, não guarda nada online. O GitHub apenas hospeda o código da app — nunca os teus dados.
👁️
Zero Knowledge
Ninguém tem acesso às tuas passwords — nem os criadores da app, nem o GitHub, nem nenhum servidor. Só tu, com a tua palavra-passe mestra, podes abrir o cofre.
📡
Funciona Offline
Após a primeira abertura, a app funciona sem internet. Podes usar em qualquer lugar, sem ligação. Os teus dados nunca precisam de estar online.
🔑
A tua Chave, o teu Cofre
A palavra-passe mestra nunca é enviada para lado nenhum. É usada localmente para encriptar e desencriptar. Se a perderes, não há recuperação — guarda-a num papel em casa!
